Installing Cordova

If you already have Cordova Installed, skip this step!

In order to compile a Phonegap and Cordova Application, you must first install the necessary applications/programs on your development machine, and the necessary libraries. 

 

Requirements:

1. NPM - https://nodejs.org/en/download/ 

2. JDK / JRK Development Kit 181 ( not higher ) - http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

3. Android Studio - https://developer.android.com/studio/

4. XCode - https://developer.apple.com/xcode/

 

Installing Cordova

On Mac, open your Terminal and add the following command: 

sudo npm install -g cordova

On Windows, open COmmand Prompt and add the following command: 

npm install -g cordova

 

Creating the App

Great! We're done installing Cordova. Now, let's continue by creating our first app. 

On both platforms,  add the following command:  

cordova create MyFirstApp com.awesomeBuyer.myFirstApp MyFirstApp

Once this command is done, let's access this folder by adding 

cd MyFirstApp

 

Adding Platforms

Now we've created the app and we're inside it. Good. Let's add platforms for iOS and Android. Be sure you've installed all necessary plugins before doing this, otherwise you'll get errors. If you get errors already, it means there is something on your machine that's causing this to happen. You can copy and paste the error code on Google to find exaclty what is causing your specific bug. 

In order to add the platforms, we have to add a specific code for each individual one. 

cordova platform add android

Excellent! We've gotten to here, now let's compile. 

Compiling your Application

To compile an app, just add the following command. 

cordova build android

 

Now you've generated an installable APK file that you can add to your device. In the compilation window you'll see the location where this is saved! 

Emulating the App.

We strongly advice NOT to test on emulators, as these do not show the performance and quality of the app as it would do on an actual device. In order to run the app on your device, directly, just plug it in your computer and add the following command 

cordova run android

If however, you do wish to emulate the app, just add the following command 

cordova emulate android

 

 

That's about it when it comes to installing and testing your first app! Let's get to the HTML part, where you add it, and what necessary elements it requires. 

 

 

 

Adding to your Project

In the app you'll find a WWW folder. This is where all the HTML files that are previously documented are edited. Follow the previous documentations to edit them, make them to your own desires and then, just compile the app! 

 

Contact Form: The contact form .php file must be hosted on your own server. You can open the contact HTML file and point to your server to asure the contact form sends messages. Testing this locally as an HTML file will return an error, we recommend compiling and testing on your actual device. 

Using Native Functionallity

In order to access any native feature of your device, you must import cordova.js in the item files. This must be imported at the end of the body, right before custom.js 

<script type="text/javascript" charset="utf-8" src="cordova.js"></script>
 
After importing this, you must add all your js code in custom.js inside the init_template() function.

Is this it?

Yeap! PhoneGap and Cordova are brilliantly easy to use. They take advantage of the power of properly built HTML, CSS and JS libraries to turn a website into an application.
